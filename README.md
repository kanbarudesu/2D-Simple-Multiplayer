A **2D side-scroller multiplayer / co-op game** built with **Unity**, featuring light competitive gameplay where players complete tasks while sabotaging each other.

The project uses **Unity Netcode for GameObjects** along with **Unity Lobby & Relay** to enable online multiplayer without direct peer-to-peer connections.

---

## ðŸ•¹ Gameplay Overview

- Each player must complete **3 mini-games / tasks**
- When a player completes a task:
  - Other players can **sabotage** it
  - Sabotaged tasks must be **recompleted**
  - The task difficulty is **reduced**
- The **first player to complete all 3 tasks** wins

This creates a mix of **co-op and competitive** gameplay.
